# with.md Editor for VS Code

A rich markdown editor for VS Code powered by [with.md](https://with.md). Open any `.md` file and edit it with a beautiful WYSIWYG interface, background themes, and real-time collaboration.

## Features

- **Rich editing** -- Format text, add headings, lists, links, and more with toolbar controls or switch to source mode
- **Real-time collaboration** -- Log in with GitHub and edit files together with your team via Hocuspocus/Yjs
- **Git integration** -- Diff against git HEAD, revert to the last committed version
- **Share snapshots** -- Generate shareable view/edit links for your markdown
- **Background themes** -- Cycle through 11 background images
- **Light/dark mode** -- Toggle between themes

## Getting Started

1. Install the extension
2. Open any `.md` or `.markdown` file
3. VS Code will offer to open it with the with.md editor

To enable real-time collaboration on a GitHub repo, sign in with GitHub when prompted.

## Configuration

| Setting | Default | Description |
|---|---|---|
| `with-md.baseUrl` | `https://with.md` | Base URL for the with.md web app |

## Requirements

- VS Code 1.85 or later
- Internet connection (the editor loads from with.md)
